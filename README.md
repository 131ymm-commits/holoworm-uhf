# HoloWorm-UHF: Universal Hierarchy Formalism for Digital Life

**The first evolving digital organism with real connectome, body physics, and a mini-universe.**

**Current Version:** HoloCosmos v0.1 (Mini-Universe with Evolution + Revolutions)

## Quick Start
```bash
pip install -r requirements.txt
python holoworm_universe_v1.py
```

Watch digital worms crawl, compete for food, evolve, and sometimes stage **revolutions**.

## Features
- Real c302 connectome (302 neurons)
- Multi-segment body physics + undulatory locomotion
- Genome + Epigenetics
- Chemotaxis
- UHF coherence law
- **Revolutions** — sudden complexity jumps against entropy
